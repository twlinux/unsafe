5 Tips for UNIX

Many operating systems including Linux and Mac are modeled after the orignal UNIX OS, which was a proprietary system developed by AT&T in the 1970s.

  • . refers to the current directory, .. refers to the parent directory.
  • The /etc directory contains system configurations. See man hier for more information.
  • The sequence ^C (ctrl-C) sends SIGINT to the foreground process, which attempts to stop it.
  • You can perform system administration remotely with the telnet or ssh protocols.
  • FTP, TFTP, and SFTP are all protocols that can be used to do file transfer (FT) from a server.
  • All these services sound great, but open ports introduce vulnerabilities.
